# 💰 예산 계산기 (Budget Calculator)

React로 만든 간단한 CRUD 예산 관리 앱입니다. 사용자는 지출 항목을 추가, 수정, 삭제하고 총 지출 금액을 확인할 수 있습니다. 
로컬 스토리지를 사용하여 새로고침해도 데이터가 유지됩니다.

## 🚀 기능
- ✅ 지출 항목 추가
- 📝 항목 수정
- 🗑️ 항목 삭제
- 🧹 전체 목록 초기화
- 💾 로컬 스토리지에 자동 저장
- 💵 총 지출 금액 자동 계산
